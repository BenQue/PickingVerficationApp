# **2\. 用户流程**

## **流程 1: 基于权限的智能导航与任务看板**

**流程图:**

graph TD  
    A\[启动应用\] \--\> B{已登录?};  
    B \--\>|否| C\[登录屏幕\];  
    B \--\>|是| G\[调用API获取用户权限\];

    C \--\> D\[输入凭证\];  
    D \--\> E\[调用认证API\];  
    E \--\> F{认证成功?};  
    F \--\>|否| C;  
    F \--\>|是| G;

    G \--\> H{权限数量?};  
    H \--\>|单一权限| I\[直接进入相应工作界面\];  
    H \--\>|多种权限| J\[进入'主导航'屏幕\];

    subgraph "核心工作区 (Home)"  
        I  
        J  
    end

    I \--\> M\[底部导航栏: \[Home\] \[我的任务\]\];  
    J \--\> M;

    M \--\> N{点击'我的任务'};  
    N \--\> O\[切换到'我的任务'屏幕\];  
    O \--\> P\[调用任务列表API\];  
    P \--\> Q\[按类型/阶段分组展示任务\];  
    Q \--\> R\[用户点击任务项\];  
    R \--\> S\[跳转至对应工作界面并强制扫描\];  
      
    O \--\> T{点击'Home'};  
    T \--\> H;
